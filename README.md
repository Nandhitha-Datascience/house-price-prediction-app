# House Price Prediction App

A machine learning-based web application developed using Python and Streamlit to predict house prices based on user inputs.

## Features
- Predicts house prices using Random Forest Regressor
- Interactive Streamlit interface
- Real-time predictions
- Data preprocessing and model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Model Performance
- R² Score: 0.84
